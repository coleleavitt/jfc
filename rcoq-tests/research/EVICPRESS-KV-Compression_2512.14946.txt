A   Pseudo Code for Algorithm


Algorithm 1: Policy Optimizer
 Input: Incoming KV Cache that needs to be stored i (size of
     the kv cache li , quality function Qi (compression ratio)),
     Storage queue T , Current Occupied size ST , Storage
     memory size C
 Output: List of operations on KV Cache: [( KV Cache i,
     compression ratio ri , operation oi )]
  1: ri ← arg max Qi (r)
  2: T .append(i, ri )
  3: L.append((i, ri , insert))
  4: ST = ST + sizeo f (li ∗ ri )
  5: while ST > C do
  6:     j, r, o, ← KV Cache in T with the least utility drop
        operation and the operation.
  7:    if o == compress then
  8:         S = ST − T j ∗ (r j − r)
  9:    else if o == evict then
 10:         r←0
 11:         S = ST − (T j ∗ r j )
 12:    end if
 13:    L.append(( j, r, o))
 14: end while
 15: return L to executor.
